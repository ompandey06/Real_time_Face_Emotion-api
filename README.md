# Real_time_Face_Emotion-api
Real-time facial emotion detection using CNN and OpenCV. Trained on the FER-2013 dataset to recognize emotions like Happy, Sad, Angry, Surprise, and more. Built with TensorFlow, Keras, and deployed via webcam feed.

The model classifies facial expressions into the following 7 emotions:
1.Angry
2.Disgust
3.Fear
4.Happy
5.Sad
6.Surprise
7.Neutral
